# service-admin

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Build for Android
1. `npm run build-release`
1. `npx cap copy`
1. `npx cap open android`
